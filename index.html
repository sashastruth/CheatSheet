Position Attributes

Tag Description
position:static Default position value - everything will appear in hoizontal blocks.
position:relative Allows you to position an element relative to its default static position on the web page.
position:absolute Will be ignored by all other elements and will be positioned relative to its closest parent element.
position:fixed Fixing an element regardless of user scrolling.
position:sticky Will remain in document flow but sticks to a specified position as the page is scrolled further.

Display Attributes

Tag Description
display:inline A box that wraps tightly around its content, cannot be altered in size by the height and width css properties, strong is inline by default.
display:block Elements that are not displayed in the same line as the content around them, they fill the width of the page and default height to accommodate their content, headings and paragraphs are block by default.
display:inline-block Combines features of both, elements can appear next to each other and can have their heights and widths specified.
